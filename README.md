## Introduction
實作Discord的NFT地板價機器人，持續將最新價格呈現給DC成員看，也可以透過指令查詢地板價，根據需求去使用，簡單幫NFT社群開發自己的功能機器人，提升社群價值

## Run

### Compile

```
tsc src/bot.ts
```

### Excute

```
node src/bot.js
```

### Combine

```
tsc src/bot.ts && node src/bot.js
```
or
```
yarn dev
```
